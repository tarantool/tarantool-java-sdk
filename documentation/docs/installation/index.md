# Введение (Загрузка артефактов библиотек)

Получить артефакты, относящиеся к коннектору, можно двумя способами:

1. @customer - данный способ позволяет вручную загрузить артефакты на локальную систему для
   тестирования и разработки. Используйте его, если в рамках вашей организации ещё не настроена
   централизованная загрузка артефактов.
2. @customer-maven (*`устарело`*) - данный способ подойдет для централизованной загрузки в хранилище
   артефактов вашей организации (Nexus, Artifactory, etc.)
3. @customer-script (*`устарело`*)
4. @maven-central - данный способ самый простой и требует наличие <<maven>>(<<gradle>>)
