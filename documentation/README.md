# Документация Tarantool Java SDK

## Локальная сборка и проверка документации

### Требования

- [docfx](https://dotnet.github.io/docfx/index.html)

### Локальная сборка и проверка проекта

- Перейти в директорию `documentation`:
    ```shell
    cd documentation
    ```
- Произвести запуск локального сервера с сайтом:
    ```shell
    ~/.dotnet/tools/docfx docfx.json --serve
    ```

### Горячее обновление исправлений

Смотри на [странице](https://dotnet.github.io/docfx/index.html#create-a-new-website)
